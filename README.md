# top-restaurant-page
The Odin Project exercise includes webpack

このエクササイズの趣旨は次のとおり
レストランのWebページというテーマのもと、
webpackの基本的な使い方を学ぶ
ES6モジュールの基本的な使い方を学ぶ

上記を意識して、モジュールの構成について検討する。
実際にECサイトの管理を想定して疑似データベース的にレストランの基本情報やメニューはそれぞれモジュールに格納してみる。
モジュールは概ね次のとおり

- index.js
- 各タブの内容
- レストランの基本情報（オブジェクト）
- メニューリスト（オブジェクトの配列）
